# 🎓 Student Performance Prediction System

An end-to-end **Machine Learning web application** built using **Python, Scikit-learn, and Streamlit** to predict student academic performance based on input features.

## 📌 Project Overview

This project predicts a student's performance (e.g., math score) based on factors such as:

- 📚 Study Hours  
- 🏫 Attendance  
- 📝 Other academic inputs  

It demonstrates a complete ML pipeline:
Data Preprocessing → Model Training → Model Saving → Web App Deployment

## 🛠️ Tech Stack

- Python

- Scikit-learn
